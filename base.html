Welcome!

I have created a Neural Network (LSTM - Long Short Term Memory Networks - a kind of Recurrent Neural Network (RNN)) to predict stock prices for the next day. Google FireBase is used as a backend, and the model updates values periodically. The web application grabs values from that backend, and displays it below. The following tools and frameworks were used for this project: Python, Google FireBase, Flask, Bootstrap, Tensorflow/Keras.
